# Loja Fit CRUD API

Projeto de estudo de QA e backend simulando uma loja de roupas fitness
com venda de bermudas e regatas.

## Tecnologias
- Node.js
- Express
- PostgreSQL (próxima etapa)
- Postman
- Cypress

## Endpoints atuais

GET /health  
GET /produtos  
GET /produtos/:id  
POST /produtos  
PUT /produtos/:id  
DELETE /produtos/:id  

## Como rodar o projeto

```bash
npm install
npm start
